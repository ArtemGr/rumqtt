A pure rust mqtt client.

[DOCUMENTATION](http://ather-energy.github.io/RuMqtt/rumqtt)
## Features

* Supports QoS 0, QoS 1 and QoS 2
* Auto reconnect
* TLS support
* Offline message buffering
* LWT
* Authentication with username and password
